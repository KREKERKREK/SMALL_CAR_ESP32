# SMALL_CAR_ESP32
Репозиторий посвящен программной части машинки для обучения на базе esp32. Машинка имеет на борту два мотора для движения, по два энкодера на каждую ось моторов, динамик, датчик расстояния, также сама esp32 оснащена Wi-Fi модулем.

Основная тестовая программа находится в папке «MainProgram», она собрана из других подпрограмм:
 AvoidingObstacles — Езда вперед с объездом препятствий если расстояние от предмета до ультразвукового датчика меньше определённого значения.
 DriveStraight — Езда прямо с корректировкой скоростей в реальном времени при помощи показаний считываемых с энкодеров, которые закреплены на осях каждого из колёс
 Music — Проигрыш зацикленной мелодии.

Остальные подпрограммы состоят из ещё более меньших фракций.
